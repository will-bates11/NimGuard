Notes:
  - Error code 6 is UC_ERR_READ_UNMAPPED: the emulator ran the first few
    instructions of the .text section and then hit a read from an address
    that has not been mapped. RSP is zero by default in a bare emulator
    context, so the first 'pop' instruction faults immediately.
  - This is expected for real-world binaries. The emulator is useful for
    testing self-contained code sequences or for validating patched bytes
    before writing them to disk (see --test-patch).
  - The emulator runs successfully when the code region is genuinely
    self-contained and does not reference external memory.
